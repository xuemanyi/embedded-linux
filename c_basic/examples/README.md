# Common C Language Example Files

These files are short and beginner-friendly.
Each file demonstrates one common C operation.

## Build
Example:
```bash
gcc 01_hello_world.c -o demo
./demo
```

## Notes
- Most examples are intentionally short.
- File read/write examples use `test.txt` in the current directory.
- These are suitable for learning, quick review, and small exercises.
